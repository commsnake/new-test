# Stylometric Analysis Project

This repository contains the output of a stylometric analysis pipeline designed to extract an author's distinct voice, tone, and syntactic fingerprint without directly quoting or distributing the original copyrighted source material.

## Included Files

- `STYLE_GUIDE.md`: A human-readable overview of the specific voice characteristics, thematic elements, lexical preferences, and syntactical rhythms observed in the text.
- `LINTING_RULES.json`: A structured format for an LLM to consume and apply specific style restrictions when mimicking the voice.
